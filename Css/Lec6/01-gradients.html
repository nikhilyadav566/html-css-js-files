Gradients:-

Gradients are a way to create smooth transitions between two or more colors. They can be used as background images or for other visual effects in web design.

Types of Gradients

  • Linear Gradients :- background: linear-gradient(to right, red, yellow);

  • Radial Gradients :- background: radial-gradient(circle, blue, green);

  • Conic Gradients :- background: conic-gradient(red, yellow, green, blue);
