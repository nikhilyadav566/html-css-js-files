Position Property Values

The CSS position property is essential for controlling the layout of elements on a web page.

1.Static

  • Description: The default positioning for all elements. Elements are placed in the normal document flow.

  • Effect on layout: Ignored in terms of top, right, bottom, and left properties.

  • Example:

                    .element {
                        position: static;
                    }

This div element has position:static
It is always positioned according to the normal flow of the page:

2.Relative

  • Description: Positioned relative to its normal position in the flow of the document.

  • Effect on layout: Moves the element without affecting surrounding elements.

  • Example:

            .element {
                position: relative;
                top: 10px;  /* Moves the element down by 10px */
                left: 20px; /* Moves the element right by 20px */
            }

This div element has position: relative
An element with position: relative; is positioned relative to its normal position:

3.Absolute

  • Description: Positioned relative to the nearest positioned ancestor (one with a position value of anything other than static).

  • Effect on layout: Removed from the document flow, can overlap other elements.

  • Example:

            .container {
                position: relative; /* Positioned ancestor */
            }
            .element {
                position: absolute;
                top: 0;
                right: 0; /* Positioned in the top-right corner 
                        of the container */
            }

This div element has position: absolute;




An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed):

4.Fixed

  • Description: Positioned relative to the viewport, meaning it stays in the same place even when the page is scrolled.
  • Effect on layout: Removed from the normal document flow.

  • Example:

            .element {
                position: fixed;
                bottom: 0;
                left: 0; 
                /* Always sticks to the bottom-left corner
                 of the viewport */
            }

This div element has position: fixed;

This div element has position: fixed; An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled:

4.Sticky

  • Description: The element is removed from the document flow and is positioned relative to the viewport, meaning it stays in the same position even when the page is scrolled.

  • Effects: It does not affect the position of other elements.

  • Example;

            .fixed-box {
                position: fixed;
                bottom: 10px; /* 10px from the bottom of the viewport */
                left: 20px; /* 20px from the left of the viewport */
              }
