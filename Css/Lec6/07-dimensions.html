Dimensions

Dimensions are used to define the size of elements on a webpages.

Key Properties

  1. Width and Height:

    • width: Sets the width of an element.

    • height:Sets the height of an element.

                        .box {
                        width: 100px;
                        height: 200px;
                    }

  2. Max and Min Dimensions:

    • max-width: Sets the maximum width of an element.
    • min-width: Sets the minimum width of an element.
    • max-height: Sets the maximum height of an element.
    • min-height: Sets the minimum height of an element.

                        .container {
                        max-width: 800px;
                        min-height: 300px;
                    }

  3. Padding and Margin:

    • padding: Space between the content and the border of an element.
    • margin: Space outside the border of an element.

                    .content {
                    padding: 20px;
                    margin: 15px;
                }

Units

  1. Absolute Units:

    • px: Pixels (fixed size).
    • cm: Centimeters (rarely used in web design).
    • mm: Millimeters (rarely used in web design).
    • in: Inches (rarely used in web design).
    • pt: Points (1 pt = 1/72 inch).

  2. Relative Units:

    • %: Percentage of the parent element's dimensions.
    • em: Relative to the font-size of the element (1em = current font size).
    • rem: Relative to the font-size of the root element (html).
    • vh: Viewport height (1% of the viewport height).
    • vw: Viewport width (1% of the viewport width).
    • vmin / vmax: The smaller or larger value of viewport width or height.
