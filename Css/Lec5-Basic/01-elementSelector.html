Element Selector

It provides a way to select elements based on their unique attributes, such as their ID, class, tag name, or other specific criteria.

Selected by name:-

p{

text-align:center ;

color : red ;

}


Selected by class

.my_first{

text-align:center;

color:white;

}


Selected by id

.my_first{

text-align:center;

color:black;

}
