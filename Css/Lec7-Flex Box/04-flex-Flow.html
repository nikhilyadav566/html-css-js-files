flex-flow:-

flex-flow property is a shorthand for setting both the flex-direction and flex-wrap properties of a flex container. It helps control the layout of flex items within the container.

Row-Wrap

box-1
box-2
box-3
box-4

Example:-

.container {
        display: flex;
        flex-flow: row wrap; /* Items in a row and will wrap onto the next line */
      }
